usage: relay-kit continuity [-h]
                            {checkpoint,rehydrate,handoff,diff-since-last} ...

Preserve and restore Relay-kit continuity across long chats and sessions.

positional arguments:
  {checkpoint,rehydrate,handoff,diff-since-last}
    checkpoint          Capture continuity snapshot
    rehydrate           Rehydrate context from last checkpoint
    handoff             Create handoff pack from latest manifest
    diff-since-last     Diff tracked continuity files since last checkpoint

options:
  -h, --help            show this help message and exit
