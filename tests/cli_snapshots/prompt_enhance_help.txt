usage: relay-kit prompt enhance [-h] --prompt PROMPT [--top-limit TOP_LIMIT]
                                [--output-file OUTPUT_FILE] [--json]
                                [project_path]

positional arguments:
  project_path          Project root to inspect

options:
  -h, --help            show this help message and exit
  --prompt PROMPT       User prompt to enhance
  --top-limit TOP_LIMIT
                        Number of route candidates to include
  --output-file OUTPUT_FILE
                        Optional prompt enhancement JSON output path
  --json                Emit machine-readable prompt enhancement
