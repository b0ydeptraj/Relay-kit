usage: relay-kit query search [-h] --query QUERY
                              [--scope {state,contracts,docs,evidence,registry}]
                              [--limit LIMIT] [--stale-days STALE_DAYS]
                              [--output-file OUTPUT_FILE] [--json]
                              [project_path]

positional arguments:
  project_path          Project root to inspect

options:
  -h, --help            show this help message and exit
  --query QUERY         Query text
  --scope {state,contracts,docs,evidence,registry}
                        Limit search to one or more scopes; defaults to all
                        scopes
  --limit LIMIT         Maximum result count
  --stale-days STALE_DAYS
                        Freshness threshold in days
  --output-file OUTPUT_FILE
                        Optional query search JSON output path
  --json                Emit machine-readable search results
