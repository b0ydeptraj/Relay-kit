usage: relay-kit support request [-h] [--severity {P0,P1,P2,P3}]
                                 [--summary SUMMARY]
                                 [--package-version PACKAGE_VERSION]
                                 [--os OPERATING_SYSTEM] [--shell SHELL]
                                 [--bundle INSTALLED_BUNDLE]
                                 [--adapter ADAPTER_TARGET]
                                 [--policy-pack POLICY_PACK]
                                 [--expected EXPECTED_BEHAVIOR]
                                 [--actual ACTUAL_BEHAVIOR]
                                 [--recent-changes RECENT_CHANGES]
                                 [--workaround WORKAROUND]
                                 [--diagnostic-file DIAGNOSTIC_FILE]
                                 [--output-file OUTPUT_FILE] [--strict]
                                 [--json]
                                 [project_path]

positional arguments:
  project_path          Project root to inspect

options:
  -h, --help            show this help message and exit
  --severity {P0,P1,P2,P3}
  --summary SUMMARY     One sentence describing the blocked workflow
  --package-version PACKAGE_VERSION
                        Relay-kit package version
  --os OPERATING_SYSTEM
                        Operating system
  --shell SHELL         Shell used for failing commands
  --bundle INSTALLED_BUNDLE
                        Installed Relay-kit bundle
  --adapter ADAPTER_TARGET
                        Adapter target such as codex or claude
  --policy-pack POLICY_PACK
                        Policy pack used for diagnostics
  --expected EXPECTED_BEHAVIOR
                        Expected behavior
  --actual ACTUAL_BEHAVIOR
                        Actual behavior
  --recent-changes RECENT_CHANGES
                        Recent Relay-kit or runtime changes
  --workaround WORKAROUND
                        Current workaround, if any
  --diagnostic-file DIAGNOSTIC_FILE
                        Diagnostic file to attach. Repeat for multiple files.
  --output-file OUTPUT_FILE
                        Output path (default: <project>/.relay-
                        kit/support/support-request.json)
  --strict              Return non-zero unless the support request is ready
  --json                Emit machine-readable support request
